# 🏫 校园打卡指南

一个展示校园热门打卡地点的趣味主题展示站，帮助同学们发现校园最美角落，记录青春美好时光。

## ✨ 功能特点

- 📍 **7个精选打卡点**：涵盖校园地标、自然风光、美食角落、休闲空间、文艺空间、特色体验
- 📁 **分类浏览**：支持按类别筛选打卡点
- 📱 **响应式设计**：适配桌面端、平板和移动端
- 🎨 **精美动画**：卡片悬停效果、弹窗交互、淡入动画
- 💫 **详情弹窗**：点击卡片查看完整介绍

## 🚀 快速开始

### 本地运行

```bash
git clone https://github.com/zyc183279-hub/campus-guide.git
cd campus-guide
# 直接打开 index.html 文件即可
```

### GitHub Pages 部署

1. 进入仓库 Settings
2. 找到 Pages 选项
3. 选择分支 `main`，点击 Save
4. 等待几分钟后访问：`https://zyc183279-hub.github.io/campus-guide/`
